立志每天完成一道OJ，以弥补此项不足。有朝一日，再写“中水平OJ进阶”。

大佬切勿见笑，菜鸡共勉！好了，开搞第1道题：

题目

描述
和所有的 OJ 平台一样，第一题作为热身题，也是送分题：给出两个非负数 aa 和 bb，输出 a+b 的结果。

输入
多组输入。

包含两个非负数 aa 和 bb，以空格分隔；aa 和 bb 保证小于 2^32 .

输出
你的输出是对一行数据处理的结果，也即 a+b 的结果。

输入样例
233 666
123 0

输出样例
899
123

思路

数字有点大，用BigDecimal。

实现

import java.math.BigDecimal;
import java.util.*;

public class Main {
    public static void main(String args[]) {
        Scanner scan = new Scanner(System.in);
        String line;
        while (scan.hasNextLine()) {
            line = scan.nextLine().trim();
            String[] array = line.split(" ");
            BigDecimal sum = new BigDecimal(0);
            for (int i = 0; i < array.length; i++) {
                BigDecimal a = new BigDecimal(array[i]);
                sum = sum.add(a);
            }
            System.out.println(sum.toBigInteger());   
        }
    }
}
